🎓 Plataforma Académica Básica

Aprende los fundamentos esenciales de la programación

1. Variables

Es un espacio en la memoria de la computadora donde guardamos un dato que puede cambiar durante la ejecución del programa.

<?php
$nombre = "Deisy"; // Las variables inician con $
?>
💡 Tip: Usa nombres descriptivos como $precio_total en lugar de solo $p.

2. Operadores Lógicos

Permiten comparar valores y tomar decisiones complejas combinando condiciones.

$edad = 20;
$tiene_licencia = true;

if ($edad >= 18 && $tiene_licencia) {
    echo "Puedes conducir";
}

3. Bucles (Loops)

Se utilizan para repetir un bloque de código varias veces mientras se cumpla una condición.

// Bucle que cuenta del 1 al 5
for ($i = 1; $i <= 5; $i++) {
    echo "Número: $i ";
}

4. Funciones

Son bloques de código reutilizables que realizan una tarea específica. Ayudan a mantener el código organizado y evitar repeticiones.

function saludar($nombre) {
    return "Hola " . $nombre;
}
💡 Tip: Si repites el mismo código más de dos veces, ¡conviértelo en una función!

5. Arreglos (Arrays)

Un arreglo es una colección de datos organizados. Imagínalo como una lista de compras donde cada elemento tiene una posición.

$frutas = ["Manzana", "Pera", "Bananos"];
echo $frutas[0]; // Muestra "Manzana"

6. Salida de Datos

Para mostrar información al usuario en PHP usamos principalmente echo o print.

echo "¡Bienvenido al mundo de la programación!";
